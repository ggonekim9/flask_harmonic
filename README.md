# flask_harmonic

感潮河段 潮汐调和分析后台算法

提供测试工具包，包括baozhen.xlsx测试数据，test.html测试页面，以及主函数app.py.
包含两个主要模块潮汐分析模块tide_model.py和水位分析模块wl_model.py
